# Problem #83: Invert A Binary Tree (Medium)

## Asked By:

This problem was asked by Google.

## Description:
 
Invert a binary tree.

## Example:

For example, given the following tree:

```
    a
   / \
  b   c
 / \  /
d   e f
```
should become:

```
  a
 / \
 c  b
 \  / \
  f e  d
```
