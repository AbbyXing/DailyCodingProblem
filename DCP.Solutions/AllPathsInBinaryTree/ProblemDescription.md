# Problem #110: Return All Paths In Binary Tree (Medium)

## Asked By:

This problem was asked by Facebook.

## Description:

Given a binary tree, return all paths from the root to leaves.

## Example:

For example, given the tree:

```
   1
  / \
 2   3
    / \
   4   5
```
Return `[[1, 2], [1, 3, 4], [1, 3, 5]]`.
